# AKQuant 中文教材优化路线图（v2）

> 本文件是教材优化的内部跟踪文档，不属于正式出版正文。
> 记录各项优化的范围、状态、来源与依赖；完成一项就更新对应状态。

## 总原则

- **段落承载思想，列表承载清单**：概念/原理/论证类内容写成段落；API 表、操作步骤、检查清单、参数表保留为列表/表格。
- **事实零漂移**：任何改写都不得新增或篡改数字、公式、API、人名与结论；逐段 diff 核对。
- **发布级示例必须实跑**：示例脚本改完真正运行（exit 0），并通过 `uvx ruff check` / `uvx ruff format --check`。
- **断网也能跑**：示例默认依赖离线合成数据，联网抓取代码放在明确标注的「联网」小节。
- **命名规范**：正文使用 AKShare、AKQuant（代码块 import 除外）。

## 关键依赖与建议执行顺序

1. **先定语气** → 解锁 Tier 1。
2. **Tier 1 逐章推进**是主线；命名规范化（④）随章顺手做，避免二次触碰同一文件。
3. **Tier 2 / Tier 3 可与 Tier 1 并行**（结构件与示例改造不依赖正文散文化）。
4. **Tier 4 / 5** 放在主体完成之后。
5. 文献核实（②）与示例实跑（Tier 3）是耗时项：优先保证**准确不杜撰、实跑 exit 0**，而非求快。
